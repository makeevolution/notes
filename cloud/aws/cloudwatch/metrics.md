# Excalidraw Data
## Text Elements
Namespace:
- A collection of related metrics

Metric:
- Time ordered set of DataPoints
- e.g. CPUUtilization

Datapoint:
- A measurement of a metric 
e.g. a server measuring its CPUUtilization will report a datapoint
- A datapoint consists of
  - timestamp
  - value

Dimension:
- Along with the datapoint, AWS also sends in a dimension
- It is a name value pair, like a metadata for the datapoint
- e.g. name=InstanceId, value=i-xxxxxxxxx
